# FinTrack 💰

Gestion des finances personnelles — fonctionne sur PC, iPhone et Android.

## Fonctionnalités
- Dépenses, budgets, comptes, wishlist, factures, analyses
- 🔄 Sync cloud en temps réel via Firebase
- 📱 PWA installable sur mobile
- 💾 Export/Import CSV

## Installation
1. Cloner ce dépôt
2. Remplacer les valeurs Firebase dans `index.html` (section `firebaseConfig`)
3. Activer GitHub Pages (Settings → Pages → main branch)
4. Ouvrir l'URL sur n'importe quel appareil
5. Sur iPhone: Safari → Partager → Sur l'écran d'accueil
6. Sur Android: Chrome → Menu → Ajouter à l'écran d'accueil
